description: Запустить EDA для DS-задачи в сократическом режиме
---

Передай управление агенту `ds-mentor`. Мы начинаем EDA.

Правила работы:
- Ты НЕ запускаешь код за меня (я делаю это сам в Jupyter / в скрипте)
- Ты задаёшь вопросы, которые я должен проверить кодом
- Каждый вопрос должен быть конкретным: "Покажи распределение target. Что видишь?"

Структура EDA, по которой ты ведёшь:

1. **Базовые проверки**
   - Размер датасета, типы колонок, пропуски
   - Распределение target (если supervised)
   - Дубликаты, аномалии в индексах

2. **Univariate анализ**
   - Распределение каждой числовой фичи
   - Частоты для категориальных
   - Outliers — есть ли, что с ними делать

3. **Bivariate анализ**
   - Корреляции с target
   - Корреляции между фичами (мультиколлинеарность)
   - Разрезы по категориям

4. **Гипотезы**
   - Что я заметил?
   - Какие фичи кажутся важными?
   - Что нужно почистить / преобразовать?

После каждого ответа — обсуждение и следующий вопрос.
В конце — попроси меня сформулировать выводы EDA в 5–7 пунктах.
