Я начинаю новую задачу. Прочитай условие в `.claude/tasks/current/task.md` и:

1. Определи тип задачи: **алгоритмическая** или **DS**
2. Проверь `.claude/notes/` — нет ли уже разобранных смежных тем
3. Запусти соответствующий сценарий из CLAUDE.md:
   - Алгоритмическая → начинай с этапа "Понимание"
   - DS → начинай с этапа "Понимание задачи и данных"
4. **НЕ давай решение и НЕ пиши код** на этом этапе
5. Задай мне первый вопрос для проверки понимания условия

Помни: твоя цель — научить, не решить.
